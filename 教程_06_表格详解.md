## 第 1 部分：表格的基本结构

### 表格是什么？

表格就像 Excel 表格一样，由**行**和**列**组成：

```
┌─────────┬─────────┬─────────┐
│ 姓名    │ 年龄    │ 城市    │  ← 第 1 行
├─────────┼─────────┼─────────┤
│ 张三    │ 25      │ 北京    │  ← 第 2 行
├─────────┼─────────┼─────────┤
│ 李四    │ 30      │ 上海    │  ← 第 3 行
└─────────┴─────────┴─────────┘
    ↑         ↑         ↑
   列1       列2       列3
```

### 三个核心标签

```html
<table>   <!-- 表格容器 -->
    <tr>  <!-- 表格行 (Table Row) -->
        <td>单元格</td>  <!-- 表格数据 (Table Data) -->
    </tr>
</table>
```

**记忆顺序：**
1. `<table>` - 先创建表格
2. `<tr>` - 然后创建行
3. `<td>` - 最后在行里创建单元格

### 最简单的表格

```html
<table>
    <tr>
        <td>单元格 1</td>
        <td>单元格 2</td>
    </tr>
    <tr>
        <td>单元格 3</td>
        <td>单元格 4</td>
    </tr>
</table>
```

**显示效果：**（没有样式的话，看起来像普通文本）
```
单元格 1  单元格 2
单元格 3  单元格 4
```

**添加边框后：**（用 CSS）
```
┌──────────┬──────────┐
│ 单元格 1 │ 单元格 2 │
├──────────┼──────────┤
│ 单元格 3 │ 单元格 4 │
└──────────┴──────────┘
```

### 代码逐行分析

```html
<table>
```
- 这是表格的**开始标签**
- 告诉浏览器："我要创建一个表格"

```html
    <tr>
```
- `<tr>` = Table Row（表格行）
- 这是第一行的开始

```html
        <td>单元格 1</td>
```
- `<td>` = Table Data（表格数据）
- 这是第一行的第一个单元格
- 内容是"单元格 1"

```html
        <td>单元格 2</td>
```
- 这是第一行的第二个单元格
- 两个 `<td>` 在同一个 `<tr>` 里，所以是同一行

```html
    </tr>
```
- 第一行结束

```html
    <tr>
        <td>单元格 3</td>
        <td>单元格 4</td>
    </tr>
```
- 第二行，同样包含两个单元格

```html
</table>
```
- 表格结束

### 一个实际的例子

```html
<table border="1">
    <tr>
        <td>姓名</td>
        <td>年龄</td>
        <td>城市</td>
    </tr>
    <tr>
        <td>张三</td>
        <td>25</td>
        <td>北京</td>
    </tr>
    <tr>
        <td>李四</td>
        <td>30</td>
        <td>上海</td>
    </tr>
</table>
```

**效果：**
```
┌──────┬──────┬──────┐
│ 姓名 │ 年龄 │ 城市 │
├──────┼──────┼──────┤
│ 张三 │  25  │ 北京 │
├──────┼──────┼──────┤
│ 李四 │  30  │ 上海 │
└──────┴──────┴──────┘
```

**注意：**
- 第一行是表头（虽然现在还用的是 `<td>`）
- 3 行 × 3 列 = 9 个单元格
- `border="1"` 添加了边框（老式写法）

## 第 2 部分：表头 &lt;th&gt;

### 什么是表头？

表头（Table Header）是表格的**标题行**，用来说明每列的含义。

### &lt;th&gt; vs &lt;td&gt;

```html
<!-- 使用 <td> -->
<tr>
    <td>姓名</td>
    <td>年龄</td>
</tr>

<!-- 使用 <th> -->
<tr>
    <th>姓名</th>
    <th>年龄</th>
</tr>
```

**区别：**

| 特性 | `<td>` | `<th>` |
|------|--------|--------|
| 用途 | 普通数据 | 表头 |
| 显示 | 普通文本 | **粗体** |
| 对齐 | 左对齐 | 居中对齐 |
| 语义 | 数据单元格 | 标题单元格 |

### 使用 &lt;th&gt; 的好处

```html
<table>
    <tr>
        <th>姓名</th>
        <th>年龄</th>
        <th>城市</th>
    </tr>
    <tr>
        <td>张三</td>
        <td>25</td>
        <td>北京</td>
    </tr>
    <tr>
        <td>李四</td>
        <td>30</td>
        <td>上海</td>
    </tr>
</table>
```

**好处：**
1. **视觉上更清晰** - 自动粗体和居中
2. **语义化** - 告诉浏览器这是标题
3. **无障碍** - 屏幕阅读器能识别
4. **SEO 友好** - 搜索引擎能理解表格结构

### 垂直表头

表头不一定在顶部，也可以在左侧：

```html
<table>
    <tr>
        <th>姓名</th>
        <td>张三</td>
    </tr>
    <tr>
        <th>年龄</th>
        <td>25</td>
    </tr>
    <tr>
        <th>城市</th>
        <td>北京</td>
    </tr>
</table>
```

**效果：**
```
┌──────┬──────┐
│ 姓名 │ 张三 │
├──────┼──────┤
│ 年龄 │  25  │
├──────┼──────┤
│ 城市 │ 北京 │
└──────┴──────┘
```

## 第 3 部分：表格分组

### 为什么要分组？

大型表格可以分为三部分：

```html
<table>
    <thead>  <!-- 表头部分 -->
        <!-- 标题行 -->
    </thead>
    
    <tbody>  <!-- 表主体部分 -->
        <!-- 数据行 -->
    </tbody>
    
    <tfoot>  <!-- 表脚部分 -->
        <!-- 总计、摘要等 -->
    </tfoot>
</table>
```

**分组的好处：**
1. **结构更清晰** - 一眼看出哪部分是什么
2. **样式更方便** - 可以给不同部分应用不同样式
3. **打印友好** - 打印时表头可以在每页重复
4. **无障碍** - 屏幕阅读器能更好地理解

### 完整示例

```html
<table>
    <!-- 表头部分 -->
    <thead>
        <tr>
            <th>商品</th>
            <th>单价</th>
            <th>数量</th>
            <th>小计</th>
        </tr>
    </thead>
    
    <!-- 表主体部分 -->
    <tbody>
        <tr>
            <td>苹果</td>
            <td>¥5</td>
            <td>3</td>
            <td>¥15</td>
        </tr>
        <tr>
            <td>香蕉</td>
            <td>¥3</td>
            <td>5</td>
            <td>¥15</td>
        </tr>
        <tr>
            <td>橙子</td>
            <td>¥4</td>
            <td>2</td>
            <td>¥8</td>
        </tr>
    </tbody>
    
    <!-- 表脚部分 -->
    <tfoot>
        <tr>
            <td colspan="3">总计</td>
            <td>¥38</td>
        </tr>
    </tfoot>
</table>
```

**效果：**
```
┌──────┬──────┬──────┬──────┐
│ 商品 │ 单价 │ 数量 │ 小计 │  ← thead（表头）
├──────┼──────┼──────┼──────┤
│ 苹果 │  ¥5  │   3  │ ¥15  │
│ 香蕉 │  ¥3  │   5  │ ¥15  │  ← tbody（数据）
│ 橙子 │  ¥4  │   2  │  ¥8  │
├──────┴──────┴──────┼──────┤
│ 总计               │ ¥38  │  ← tfoot（总计）
└────────────────────┴──────┘
```

### 注意事项

1. **`<tfoot>` 的位置**：
   - 在 HTML 中可以写在 `<tbody>` 前面
   - 浏览器会自动把它显示在底部

2. **不是必须的**：
   - 小表格可以不分组
   - 但大表格强烈建议分组

## 第 4 部分：合并单元格

### colspan - 横向合并（跨列）

**用途：**让一个单元格占据多列的空间。

```html
<table>
    <tr>
        <td colspan="2">这个单元格占 2 列</td>
    </tr>
    <tr>
        <td>列 1</td>
        <td>列 2</td>
    </tr>
</table>
```

**效果：**
```
┌──────────────────┐
│ 这个单元格占 2 列 │  ← 合并了 2 列
├─────────┬────────┤
│  列 1   │  列 2  │
└─────────┴────────┘
```

**解释：**
- `colspan="2"` 表示这个单元格横向跨越 2 列
- 第一行只需要 1 个 `<td>`，而不是 2 个
- 第二行仍然有 2 个 `<td>`

### rowspan - 纵向合并（跨行）

**用途：**让一个单元格占据多行的空间。

```html
<table>
    <tr>
        <td rowspan="2">这个单元格占 2 行</td>
        <td>行 1, 列 2</td>
    </tr>
    <tr>
        <td>行 2, 列 2</td>
    </tr>
</table>
```

**效果：**
```
┌──────────────┬─────────────┐
│              │ 行 1, 列 2  │
│ 这个单元格   ├─────────────┤
│   占 2 行    │ 行 2, 列 2  │
└──────────────┴─────────────┘
```

**解释：**
- `rowspan="2"` 表示这个单元格纵向跨越 2 行
- 第一行有 2 个 `<td>`
- 第二行只有 1 个 `<td>`（因为第一个被合并了）

### 同时使用 colspan 和 rowspan

```html
<table>
    <tr>
        <td rowspan="2" colspan="2">占 2 行 2 列</td>
        <td>A</td>
    </tr>
    <tr>
        <td>B</td>
    </tr>
    <tr>
        <td>C</td>
        <td>D</td>
        <td>E</td>
    </tr>
</table>
```

**效果：**
```
┌──────────────────┬───┐
│                  │ A │
│  占 2 行 2 列    ├───┤
│                  │ B │
├─────┬──────┬─────┴───┤
│  C  │  D   │    E    │
└─────┴──────┴─────────┘
```

### 常见错误

#### 错误 1：列数不匹配

```html
<!-- ❌ 错误 -->
<table>
    <tr>
        <td colspan="3">标题</td>
    </tr>
    <tr>
        <td>A</td>
        <td>B</td>
        <!-- 缺少一个 <td> -->
    </tr>
</table>
```

**问题：**第一行说有 3 列，第二行却只有 2 个单元格。

**正确：**
```html
<!-- ✅ 正确 -->
<table>
    <tr>
        <td colspan="3">标题</td>
    </tr>
    <tr>
        <td>A</td>
        <td>B</td>
        <td>C</td>  <!-- 补上第三个 -->
    </tr>
</table>
```

#### 错误 2：忘记考虑 rowspan

```html
<!-- ❌ 错误 -->
<table>
    <tr>
        <td rowspan="2">跨 2 行</td>
        <td>A</td>
    </tr>
    <tr>
        <td>B</td>
        <td>C</td>  <!-- 多了一个！ -->
    </tr>
</table>
```

**问题：**第一列已经被 rowspan 占据，第二行不应该再有它。

**正确：**
```html
<!-- ✅ 正确 -->
<table>
    <tr>
        <td rowspan="2">跨 2 行</td>
        <td>A</td>
    </tr>
    <tr>
        <td>B</td>  <!-- 不要写第一列，它被上面合并了 -->
    </tr>
</table>
```

## 第 6 部分：表格属性

### 常用的表格属性

| 属性 | 用途 | 示例 | 状态 |
|------|------|------|------|
| `border` | 设置边框 | `<table border="1">` | ⚠️ 已废弃 |
| `width` | 设置宽度 | `<table width="100%">` | ⚠️ 已废弃 |
| `cellpadding` | 单元格内边距 | `<table cellpadding="10">` | ❌ 已废弃 |
| `cellspacing` | 单元格间距 | `<table cellspacing="0">` | ❌ 已废弃 |

### 单元格属性

| 属性 | 用途 | 示例 | 状态 |
|------|------|------|------|
| `colspan` | 横向合并 | `<td colspan="2">` | ✅ 使用 |
| `rowspan` | 纵向合并 | `<td rowspan="3">` | ✅ 使用 |
| `align` | 水平对齐 | `<td align="center">` | ⚠️ 已废弃 |
| `valign` | 垂直对齐 | `<td valign="middle">` | ⚠️ 已废弃 |

### ⚠️ 重要提示

很多老的表格属性已被废弃，现代网页应该使用 **CSS** 来设置样式！

```html
<!-- ❌ 老方法（不推荐） -->
<table border="1" cellpadding="10" cellspacing="0" width="100%">
    <tr>
        <td align="center" valign="middle">内容</td>
    </tr>
</table>

<!-- ✅ 新方法（推荐） -->
<table style="border: 1px solid #ddd; width: 100%; border-collapse: collapse;">
    <tr>
        <td style="padding: 10px; text-align: center; vertical-align: middle;">内容</td>
    </tr>
</table>
```

### 常用的 CSS 样式

```css
/* 表格样式 */
table {
    border-collapse: collapse;  /* 合并边框 */
    width: 100%;                /* 宽度 */
    margin: 1rem 0;             /* 外边距 */
}

/* 单元格样式 */
td, th {
    border: 1px solid #ddd;     /* 边框 */
    padding: 10px;              /* 内边距 */
    text-align: left;           /* 水平对齐 */
}

/* 表头样式 */
th {
    background-color: #4CAF50;  /* 背景色 */
    color: white;               /* 文字颜色 */
    font-weight: bold;          /* 粗体 */
}

/* 斑马纹（隔行变色） */
tr:nth-child(even) {
    background-color: #f9f9f9;
}

/* 鼠标悬停效果 */
tr:hover {
    background-color: #f0f7ff;
}
```

## 第 7 部分：最佳实践

### ✅ 好的做法

#### 1. 使用语义化标签

```html
<!-- ✅ 好 -->
<table>
    <thead>
        <tr><th>标题</th></tr>
    </thead>
    <tbody>
        <tr><td>数据</td></tr>
    </tbody>
    <tfoot>
        <tr><td>总计</td></tr>
    </tfoot>
</table>

<!-- ❌ 差 -->
<table>
    <tr><th>标题</th></tr>
    <tr><td>数据</td></tr>
    <tr><td>总计</td></tr>
</table>
```

#### 2. 使用 &lt;th&gt; 作为表头

```html
<!-- ✅ 好 -->
<tr>
    <th>姓名</th>
    <th>年龄</th>
</tr>

<!-- ❌ 差 -->
<tr>
    <td><strong>姓名</strong></td>
    <td><strong>年龄</strong></td>
</tr>
```

#### 3. 用 CSS 控制样式

```html
<!-- ✅ 好 -->
<table style="border-collapse: collapse; width: 100%;">
    <tr>
        <td style="border: 1px solid #ddd; padding: 8px;">内容</td>
    </tr>
</table>

<!-- ❌ 差 -->
<table border="1" cellpadding="8" width="100%">
    <tr>
        <td>内容</td>
    </tr>
</table>
```

#### 4. 保持代码整洁

```html
<!-- ✅ 好：缩进清晰 -->
<table>
    <thead>
        <tr>
            <th>标题</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>数据</td>
        </tr>
    </tbody>
</table>

<!-- ❌ 差：没有缩进 -->
<table>
<thead>
<tr>
<th>标题</th>
</tr>
</thead>
</table>
```

### ❌ 避免的做法

#### 1. 不要用表格布局网页

```html
<!-- ❌ 错误：用表格布局 -->
<table>
    <tr>
        <td>导航栏</td>
    </tr>
    <tr>
        <td>
            <table>
                <tr>
                    <td>侧边栏</td>
                    <td>主内容</td>
                </tr>
            </table>
        </td>
    </tr>
    <tr>
        <td>页脚</td>
    </tr>
</table>

<!-- ✅ 正确：用 HTML5 + CSS 布局 -->
<header>导航栏</header>
<main>
    <aside>侧边栏</aside>
    <article>主内容</article>
</main>
<footer>页脚</footer>
```

**原因：**
- 表格是用来展示**数据**的，不是用来**布局**的
- 布局应该用 CSS（flexbox, grid）
- 用表格布局会导致代码混乱、难以维护、性能差

#### 2. 不要嵌套太多层表格

```html
<!-- ❌ 错误：表格套表格套表格 -->
<table>
    <tr>
        <td>
            <table>
                <tr>
                    <td>
                        <table>
                            <tr><td>数据</td></tr>
                        </table>
                    </td>
                </tr>
            </table>
        </td>
    </tr>
</table>
```

**问题：**
- 代码混乱，难以维护
- 性能差
- 屏幕阅读器难以理解

#### 3. 不要忘记表头

```html
<!-- ❌ 差：没有表头 -->
<table>
    <tr>
        <td>张三</td>
        <td>25</td>
        <td>北京</td>
    </tr>
</table>
<!-- 用户不知道这三列分别是什么 -->

<!-- ✅ 好：有表头 -->
<table>
    <tr>
        <th>姓名</th>
        <th>年龄</th>
        <th>城市</th>
    </tr>
    <tr>
        <td>张三</td>
        <td>25</td>
        <td>北京</td>
    </tr>
</table>
```

#### 4. 不要让表格过宽

```html
<!-- ✅ 好：在窄屏幕上可以滚动 -->
<div style="overflow-x: auto;">
    <table>
        <!-- 表格内容 -->
    </table>
</div>
```

## 第 8 部分：响应式表格

### 问题

表格在**手机等小屏幕**上容易显示不全：

```
手机屏幕很窄
┌─┬─┬─┬─┬─┬─┬─┐
│姓│年│城│职│邮│电│...  ← 列太多，显示不全
```

### 解决方案

#### 方案 1：横向滚动

```html
<div style="overflow-x: auto;">
    <table>
        <!-- 表格内容 -->
    </table>
</div>
```

**好处：**
- 简单易实现
- 保留完整信息

**缺点：**
- 需要手动滑动

#### 方案 2：隐藏不重要的列

```css
/* 在小屏幕上隐藏某些列 */
@media screen and (max-width: 600px) {
    .hide-mobile {
        display: none;
    }
}
```

```html
<table>
    <tr>
        <th>姓名</th>
        <th>年龄</th>
        <th class="hide-mobile">城市</th>
        <th class="hide-mobile">职业</th>
    </tr>
</table>
```

#### 方案 3：改变布局

在小屏幕上把表格变成列表形式（需要 CSS 和 JavaScript）。

## 常见问题（FAQ）

### Q1: 表格一定要有边框吗？

**A:** 不一定！边框只是为了让表格更清晰。

```html
<!-- 有边框 -->
<table border="1">...</table>

<!-- 无边框 -->
<table>...</table>
```

很多现代设计使用**轻微的背景色**或**细线**，而不是粗边框。

### Q2: 每行的单元格数量必须相同吗？

**A:** 一般情况下是的，但可以用 `colspan` 和 `rowspan` 来调整。

```html
<!-- ✅ 合法：使用 colspan -->
<table>
    <tr>
        <td colspan="2">占 2 列</td>
    </tr>
    <tr>
        <td>1</td>
        <td>2</td>
    </tr>
</table>

<!-- ❌ 错误：单元格数量不匹配 -->
<table>
    <tr>
        <td>1</td>
    </tr>
    <tr>
        <td>1</td>
        <td>2</td>
    </tr>
</table>
```

### Q3: `<thead>`、`<tbody>`、`<tfoot>` 必须都有吗？

**A:** 不必须，但大表格建议使用。

```html
<!-- 简单表格 -->
<table>
    <tr><th>标题</th></tr>
    <tr><td>数据</td></tr>
</table>

<!-- 复杂表格 -->
<table>
    <thead><tr><th>标题</th></tr></thead>
    <tbody><tr><td>数据</td></tr></tbody>
    <tfoot><tr><td>总计</td></tr></tfoot>
</table>
```

### Q4: 如何让表格居中？

**A:** 使用 CSS：

```html
<table style="margin: 0 auto;">
    <!-- 表格内容 -->
</table>
```

或者：

```html
<div style="text-align: center;">
    <table style="display: inline-block;">
        <!-- 表格内容 -->
    </table>
</div>
```

### Q5: 如何让数字右对齐？

**A:** 使用 CSS：

```html
<td style="text-align: right;">123.45</td>
```

或者给数字列加一个 class：

```css
.number {
    text-align: right;
}
```

```html
<td class="number">123.45</td>
```
