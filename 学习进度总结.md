## 💡 重要概念回顾

### 1. 什么是 HTML？
**HTML** = HyperText Markup Language（超文本标记语言）
- 用来创建网页的**结构**
- 用标签来**标记内容**的含义
- 告诉浏览器**如何组织和显示**内容

### 2. 标签和属性
```html
<标签名 属性名="属性值">内容</标签名>
<a href="https://baidu.com">链接</a>
```

### 3. 语义化的重要性
- 帮助搜索引擎理解内容
- 帮助屏幕阅读器（给盲人用）
- 使代码更易阅读和维护
- 提高网页的可访问性

### 4. HTML 的三个层面

| 层面 | 作用 | 完成度 |
|------|------|--------|
| **结构** (HTML) | 定义内容的结构和含义 | ✅ 学中 |
| **样式** (CSS) | 控制网页的外观（颜色、字体等） | ⏳ 待学 |
| **交互** (JavaScript) | 添加网页的交互功能 | ⏳ 待学 |

## 📝 常见问题解答

### Q: HTML 和 HTML5 的区别是什么？
A: HTML5 是 HTML 的最新版本（2014年标准化）。现在说的"HTML"通常就是指 HTML5。旧版本的 HTML（HTML 4.01）已经过时了。

### Q: `<b>` 和 `<strong>` 在视觉上看起来一样，为什么要区分？
A: 虽然视觉上都是加粗，但：
- `<strong>` 告诉搜索引擎和屏幕阅读器："这很重要"
- `<b>` 只是"让这个字看起来粗一点"

搜索引擎会更重视被 `<strong>` 标记的内容，屏幕阅读器会改变语调。

### Q: 为什么要用 `<h1>` 不用 `<b>`？
A: 虽然都能让文字变大，但：
- `<h1>` 是语义化的，表示"这是最重要的标题"
- `<b>` 只是样式，没有含义
- 使用 `<h1>` 让网页结构清晰

### Q: 我应该在多少标签时停止？
A: 使用**足够表达意思的最少标签**。不要过度标记。

```html
✅ 好：<p>这是 <strong>很重要的</strong> 信息。</p>
❌ 坏：<p>这是 <strong>很<em>重要</em>的</strong> 信息。</p>
```

## ✨ 学习建议

1. **动手实践** - 不要仅仅阅读，要在电脑上操作
