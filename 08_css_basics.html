第 1 部分：CSS 的 3 种使用方式

三种方式：
  • 内联样式（inline）：直接写在元素上，如 style="color:red"
  • 内部样式（internal）：写在页面的 <style> 标签中（本页采用）
  • 外部样式（external）：写在 .css 文件中，通过 <link> 引入
<!-- 内联样式（仅演示） --> <h2 style="color:red">Hello CSS</h2> <!-- 内部样式（本页 head 内） --> <style> h2 { color: #667eea; } </style> <!-- 外部样式（推荐在真实项目中） --> <link rel="stylesheet" href="styles.css" />

真实项目中更推荐“外部样式”，便于复用与维护。

第 2 部分：基础选择器

ID 选择器（#id）

ID 在页面中应该唯一，适合少量特定元素。

类选择器（.class）

类可以重复使用，适合批量样式复用。

这是被类选择器选中的段落（背景为浅黄）。

/* CSS */ #unique-title { color: #10b981; } .highlight { background: #fffbeb; } /* 类型选择器：直接选标签 */ p { color: #374151; }

第 3 部分：组合与层级选择器

后代选择器（空格）

.card p 选中 card 中的所有段落。

card descendant

子代选择器（>）

仅选中直接子元素：.card > h4

并列选择器（,）

一条规则可应用到多个选择器。

第 4 部分：属性选择器

根据属性和值选择元素，常见于表单与链接：

新窗口打开的链接

a[target="_blank"] { color: #2563eb } input[placeholder^="请输入"] { background: #eff6ff } input[type="email"] { border: 1px solid #60a5fa }

第 6 部分：盒模型（Box Model）

盒模型由内容、内边距（padding）、边框（border）、外边距（margin）组成。

内容区域
content-box（对比）
.box { width: 160px; padding: 12px; border: 3px solid; margin: 10px; } /* 推荐：border-box 更易计算总尺寸 */ .box { box-sizing: border-box }

第 8 部分：层叠与优先级（Specificity）

选择器优先级：内联 > ID > 类/属性/伪类 > 标签。

普通文本

强调文本（类选择器）

被 !important 强制的文本

/* 类选择器 */ .emphasis { color: #ef4444 } /* ID 优先级更高 */ #priority-strong .emphasis { color: #10b981 } /* !important 会强制覆盖（请谨慎使用） */ .force { color: #6366f1 !important }
最佳实践：尽量通过更合理的选择器与结构组织样式，避免滥用 !important。
