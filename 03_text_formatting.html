文本格式化完全教程

学习如何在网页中美化和格式化文字

1. 标题标签（h1 - h6）

标题按照从大到小的顺序：

这是 h1 - 一级标题（最大）

这是 h2 - 二级标题

这是 h3 - 三级标题

这是 h4 - 四级标题

这是 h5 - 五级标题
这是 h6 - 六级标题（最小）

规则： 一个页面通常只有一个 h1，其他标题用于内容分层。

2. 段落标签（p）

这是第一个段落。段落标签会自动在文本前后添加空白行，使内容更易阅读。

这是第二个段落。如果你在 HTML 代码中写了多行文字，它们仍然会被当作一个段落。

这是第三个段落。段落是网页中最常见的内容容器。

技巧： 始终使用 <p> 标签来包裹段落文本，这样更符合语义化原则。

3. 加粗文本

方法 1：<strong> 标签（有语义）

这是一句 非常重要 的文字。

说明： <strong> 表示内容很重要，搜索引擎和屏幕阅读器会特别对待。

方法 2：<b> 标签（无语义）

这是一句 加粗 的文字。

说明： <b> 只是样式，不表示重要性。

对比：应该何时使用？

✅ 使用 <strong> 当你要表示内容很重要

例如：警告：不要删除这个文件！

✅ 使用 <b> 当你只是想让文字看起来加粗，没有特殊含义

例如：摘要： 这篇文章讲述了...

4. 斜体文本

方法 1：<em> 标签（有语义 - 强调）

这个词有 特殊的强调 含义。

说明： <em> 表示"emphasis"（强调），用于强调某个部分。

方法 2：<i> 标签（无语义）

书名：活着

说明： <i> 只是样式，通常用于书名、外文、技术术语等。

实际用法

我最喜欢的书是 三体，它 深深地 影响了我对宇宙的看法。

5. 删除线和标记

<del> 标签（删除的文本）

原价：¥999

现价：¥599

说明： <del> 表示这部分内容已被删除，常用于显示价格变化。

<s> 标签（不再准确的文本）

状态：进行中 已完成

说明： <s> 表示这部分不再准确或有效。

<mark> 标签（高亮标记）

这是一段长文本，其中 这部分被高亮 了。

说明： <mark> 用于标记重要或相关的文本。

6. 下划线 <u>

这是 下划线文本。

注意： 下划线容易与链接混淆，所以在网页中应谨慎使用。

7. 上标和下标

上标 <sup>

数学公式：E=mc2

版权标记：Smart®

下标 <sub>

化学公式：H2O

化学方程式：C6H12O6

8. 代码和计算机输入

<code> 标签（代码）

使用 console.log() 来输出信息。

说明： <code> 用于标记程序代码或命令。

<kbd> 标签（键盘输入）

按下 Ctrl + S 保存文件。

说明： <kbd> 用于表示用户按下的按键。

<samp> 标签（程序输出）

运行命令后，终端会显示：Hello, World!

说明： <samp> 用于表示程序的输出结果。

<pre> 标签（预格式化文本）

function sayHello() {
    console.log("Hello");
    return true;
}

说明： <pre> 保留空格和换行，常用于显示代码块。

9. 引用

<q> 标签（短引用）

有句名言：生活就像骑自行车，为了保持平衡，你必须不断前进。

说明： <q> 用于短的引用文本，通常会自动添加引号。

<blockquote> 标签（长引用）

这是一个长的引用块。通常用于引用他人的较长文字或重要段落。

引用通常会自动缩进，以区别于普通文本。

— 来自《某本书》

10. 换行和分割线

<br> 标签（换行）

这是第一行
这是第二行
这是第三行

说明： <br> 用于强制换行，但一般用 CSS 更好。

<hr> 标签（水平分割线）

这是前面的内容

这是后面的内容

说明： <hr> 用于分割不同的主题或内容块。

11. 缩写和定义

<abbr> 标签（缩写）

HTML 是网页的标准标记语言。

CSS 用于设计网页样式。

说明： 鼠标悬停在缩写上时会显示完整的说明文本。

<dfn> 标签（定义）

HTML 是超文本标记语言，是创建网页的基础。

说明： <dfn> 用于标记首次定义的术语。
