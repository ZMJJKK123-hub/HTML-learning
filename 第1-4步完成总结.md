## 🏆 你已经掌握的关键概念

### 1. **语义化的重要性**
```html
❌ <div>标题</div>
✅ <h1>标题</h1>

❌ <div>我的网站</div>
✅ <header><h1>我的网站</h1></header>

❌ <span>很重要</span>
✅ <strong>很重要</strong>
```

### 2. **属性的使用**
```html
<标签 属性1="值1" 属性2="值2">内容</标签>

例子：
<a href="url" target="_blank" title="提示">链接</a>
<img src="path" alt="描述" width="300">
```

### 3. **路径理解**
```
相对路径：image.jpg, images/photo.jpg, ../images/photo.jpg
绝对路径：/images/photo.jpg, https://example.com/photo.jpg
```

### 4. **链接的强大之处**
```html
<!-- 网站间的连接 -->
<a href="https://google.com">Google</a>

<!-- 网站内的导航 -->
<a href="about.html">关于</a>

<!-- 同页面内的跳转 -->
<a href="#top">回到顶部</a>

<!-- 特殊功能 -->
<a href="mailto:user@example.com">邮件</a>
<a href="tel:+86138...">电话</a>
```

### 5. **图片的正确使用**
```html
<img src="photo.jpg" 
     alt="具体的图片描述" 
     width="400"
     title="鼠标悬停提示">
```

**1. 个人简历网站**
```html

<html lang="zh-CN">

    <meta charset="UTF-8">
    <title>张三 - Web 开发工程师</title>

<body>

        <h1>张三</h1>
        <p>Web 开发工程师</p>
    </header>

        <a href="#skills">技能</a>

                <li>CSS</li>
                <li>JavaScript</li>
            </ul>
        </section>

        <section id="projects">
            <h2>项目</h2>
            <article>
                <h3>项目 1：个人博客</h3>
                <p>用 HTML 和 CSS 构建的个人博客</p>

        <section id="contact">
```

## 🎓 为什么这些知识很重要？

### 1. 语义化标签
- 帮助搜索引擎理解你的网页
- 帮助屏幕阅读器为视障用户读出内容
- 让代码更易维护
- 提高网页的可访问性

### 2. 正确的链接
- 连接整个互联网
- 导航系统的基础
- SEO（搜索引擎优化）的关键

### 3. 图片处理
- 提供替代文本（alt）是无障碍设计的基础
- 正确的路径避免加载失败
- 合适的图片格式优化网页性能

### 4. 文本格式化
- 区分内容的层级和重要性
- 提高用户体验
- 改善可读性
