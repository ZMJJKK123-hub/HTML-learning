# 第 4 步：链接和图片详解 - 让网页连接起来

## 📌 文件说明
这个文件展示了如何在网页中添加**链接和图片**。链接是网页的灵魂，它们连接了整个互联网。图片让网页更加丰富有趣。

---

## 第一部分：链接 `<a>` 标签

### 为什么链接很重要？

想象一个没有链接的网站：
- ❌ 用户无法点击查看其他页面
- ❌ 无法跳转到其他网站
- ❌ 就像一个孤立的网页

有了链接：
- ✅ 用户可以在不同页面间导航
- ✅ 可以连接到其他网站
- ✅ 网站变成一个完整的系统

---

## 🔍 链接的完整讲解

### 1. 最基础的链接：`href` 属性

```html
<a href="https://www.baidu.com">点击访问百度</a>
```

**分解：**
- `<a>` - 锚点标签
- `href="https://www.baidu.com"` - 属性，指定链接的目标地址
- `点击访问百度` - 用户看到的链接文本
- `</a>` - 闭标签

**什么是 href？**
- href = HyperText REFerence（超文本引用）
- 这个属性告诉浏览器点击链接时应该打开哪个地址

**真实场景：**
```html
<!-- 导航菜单 -->
<nav>
    <a href="https://www.amazon.com">亚马逊</a>
    <a href="https://www.ebay.com">eBay</a>
    <a href="https://www.aliexpress.com">速卖通</a>
</nav>
```

---

### 2. 页面内部链接（锚点跳转）

这是一个强大的功能：让用户可以跳转到同一页面的不同部分。

**工作原理：**

```html
<!-- 1. 导航菜单 -->
<nav>
    <a href="#about">关于我</a>
    <a href="#skills">技能</a>
    <a href="#contact">联系方式</a>
</nav>

<!-- 2. 页面中的目标位置 -->
<section id="about">
    <h2>关于我</h2>
    <p>我是一名 Web 开发者...</p>
</section>

<section id="skills">
    <h2>技能</h2>
    <p>我掌握的技能包括...</p>
</section>

<section id="contact">
    <h2>联系方式</h2>
    <p>邮箱：example@example.com</p>
</section>
```

**过程：**
1. 用户点击 "关于我" 链接
2. 浏览器看到 `href="#about"`
3. 浏览器查找 `id="about"` 的元素
4. 页面自动滚动到那个位置

**关键点：**
- `#` 符号表示这是页面内的跳转
- `id` 必须是唯一的（每个页面内只能有一个相同的 id）
- `id` 和 `href` 的值必须完全匹配

---

### 3. 在新标签页打开链接

```html
<a href="https://www.google.com" target="_blank">在新标签页打开 Google</a>
```

**`target="_blank"` 的含义：**
- "新建一个空白（blank）的标签页"
- 原来的页面保持打开
- 用户可以随时回到原来的页面

**什么时候使用？**
- ✅ 链接到外部网站
- ✅ 不想让用户离开你的网页
- ❌ 不要过度使用（太烦人）

**其他 target 值：**
```html
target="_self"    <!-- 在当前窗口打开（默认） -->
target="_blank"   <!-- 在新标签页打开 -->
target="_parent"  <!-- 在父框架打开 -->
target="_top"     <!-- 在顶层窗口打开 -->
```

---

### 4. 特殊的链接：邮件

```html
<a href="mailto:example@example.com">发送邮件给我</a>
```

**工作原理：**
- 用户点击链接
- 浏览器打开用户的**默认邮箱应用**（如 Outlook、Gmail）
- 邮件地址自动填入 "收件人" 字段

**实际例子：**
```html
<!-- 一个完整的联系方式示例 -->
<p>有任何问题，请：</p>
<ul>
    <li><a href="mailto:support@example.com">发送邮件</a></li>
    <li><a href="tel:+8613800138000">拨打电话</a></li>
    <li><a href="#contact">查看联系表单</a></li>
</ul>
```

---

### 5. 电话链接

```html
<a href="tel:+8613800138000">拨打我的电话</a>
```

**工作原理：**
- 在**手机**上：点击链接会拨打电话
- 在**电脑**上：通常没有反应（除非安装了 Skype 等应用）

**格式：**
```
tel:+86 10 1234 5678    （国内格式）
tel:+86-10-1234-5678    （另一种格式）
```

**为什么要这样做？**
- 提高移动端用户的体验
- 用户不需要手动输入电话号码

---

### 6. 其他常用的 href 值

```html
<!-- 相对路径（同一个网站内） -->
<a href="about.html">关于页面</a>

<!-- 绝对路径（从网站根目录开始） -->
<a href="/pages/about.html">关于页面</a>

<!-- 下载文件 -->
<a href="/files/document.pdf" download>下载 PDF</a>

<!-- 调用脚本（高级） -->
<a href="javascript:void(0)" onclick="doSomething()">执行操作</a>
```

---

### 7. 链接的其他属性

#### `title` 属性（提示文本）

```html
<a href="https://www.wikipedia.org" title="维基百科 - 自由的百科全书">
    维基百科
</a>
```

**效果：** 用户鼠标悬停时看到提示

#### `rel` 属性（关系描述）

```html
<!-- 告诉搜索引擎不追踪这个链接 -->
<a href="https://spam-site.com" rel="nofollow">不要追踪的链接</a>

<!-- 用于样式链接 -->
<a href="styles.css" rel="stylesheet">样式表</a>

<!-- 前后导航 -->
<a href="next.html" rel="next">下一页</a>
<a href="prev.html" rel="prev">上一页</a>
```

#### `download` 属性

```html
<!-- 点击时下载文件而不是打开 -->
<a href="image.jpg" download>下载图片</a>

<!-- 指定下载时的文件名 -->
<a href="image.jpg" download="my-photo.jpg">下载图片</a>
```

## 第二部分：图片 `<img>` 标签

### 为什么要学图片？

一张图片胜过千言万语：
- 网页吸引力大幅提升
- 用户更容易理解内容
- 提供视觉上的休息

---

## 🔍 图片的完整讲解

### 1. 最基础的图片

```html
<img src="image.jpg" alt="图片描述">
```

**分解：**
- `<img>` - 图片标签（自闭合，没有 `</img>`）
- `src="image.jpg"` - 图片的路径（必需）
- `alt="图片描述"` - 图片加载失败时显示的文本（必需）

**重要：** `src` 和 `alt` 都是**必需的**，不能省略。

---

### 2. `src` 属性（图片路径）

#### 本地文件：

```html
<!-- 同一文件夹 -->
<img src="photo.jpg" alt="照片">

<!-- 子文件夹 -->
<img src="images/photo.jpg" alt="照片">

<!-- 上级文件夹 -->
<img src="../photos/photo.jpg" alt="照片">
```

#### 网络地址：

```html
<!-- 完整 URL -->
<img src="https://example.com/image.jpg" alt="照片">

<!-- 协议相对 URL（自动选择 http 或 https） -->
<img src="//example.com/image.jpg" alt="照片">
```

---

### 3. `alt` 属性（替代文本）- 非常重要！

**什么是 alt？**
- alt = alternative（替代）
- 当图片无法显示时，显示这段文字

**为什么重要？**

1. **可访问性** - 屏幕阅读器会读出 alt 文本
   - 盲人用户可以通过听到 alt 文本来理解图片

2. **用户体验** - 图片加载失败时
   - 网络慢、图片删除、文件名错误等原因

3. **搜索引擎优化** - 搜索引擎通过 alt 了解图片内容
   - 帮助网站排名

4. **语义化** - 说明这张图片是关于什么的

**好的 alt 文本示例：**

```html
<!-- ❌ 不好 -->
<img src="pic.jpg" alt="图片">

<!-- ❌ 不好 -->
<img src="photo.jpg" alt="">

<!-- ✅ 好 -->
<img src="beach.jpg" alt="白色沙滩和蓝色的海水，远处有帆船">

<!-- ✅ 好（对于装饰性图片） -->
<img src="decoration.jpg" alt="">

<!-- ✅ 好（对于图表） -->
<img src="sales-chart.png" alt="2024年销售数据：第一季度50万，第二季度75万，第三季度90万">
```

**规则：**
- 描述图片的内容，不需要说"这是一张图片"
- 尽量简洁但信息完整
- 对于装饰性图片，可以留空 `alt=""`
- 对于复杂图表，在旁边的文本中详细说明

---

### 4. `width` 和 `height` 属性

```html
<img src="photo.jpg" 
     alt="照片"
     width="300" 
     height="200">
```

**作用：**
- 指定图片在页面上显示的大小（单位：像素）
- 浏览器知道图片的大小后，可以预留空间，防止页面抖动

**使用建议：**
```html
<!-- ✅ 好的做法：只指定宽度，让浏览器自动计算高度 -->
<img src="photo.jpg" alt="照片" width="300">

<!-- ❌ 避免：同时指定宽高，可能导致图片变形 -->
<img src="photo.jpg" alt="照片" width="400" height="200">
  <!-- 如果原图是 800x600，现在变成了 400x200，会被拉伸 -->
```

**计算高度的方法：**
```
原图尺寸：800 x 600
我们想要宽度：300

高度 = 300 × (600 ÷ 800) = 300 × 0.75 = 225

所以设置：width="300" （高度会自动变成 225）
```

---

### 5. `title` 属性（鼠标悬停提示）

```html
<img src="photo.jpg" 
     alt="照片"
     title="这是我在海边拍的照片，2024年夏天">
```

**效果：** 鼠标悬停时显示提示文本

---

### 6. 图片格式讲解

| 格式 | 扩展名 | 用途 | 优点 | 缺点 |
|------|--------|------|------|------|
| **JPEG** | .jpg, .jpeg | 照片、复杂图像 | 文件小，支持各种质量设置 | 有损压缩，不支持透明 |
| **PNG** | .png | 图标、logo、需要透明的图 | 无损压缩，支持透明 | 文件比 JPEG 大 |
| **GIF** | .gif | 简单图像、动画 | 支持动画 | 颜色有限（最多256色） |
| **WebP** | .webp | 现代网站 | 比 JPEG 和 PNG 都小 | 老版本浏览器不支持 |
| **SVG** | .svg | 矢量图、图标 | 可无限缩放，文件小 | 不适合照片 |

**选择指南：**

```html
<!-- 照片 → 用 JPEG -->
<img src="vacation.jpg" alt="度假照片">

<!-- logo、图标、需要透明 → 用 PNG -->
<img src="logo.png" alt="公司 logo">

<!-- 动画 → 用 GIF -->
<img src="loading.gif" alt="加载中">

<!-- 现代网站推荐 → 用 WebP 或 PNG -->
<img src="image.webp" alt="图片">
```

---

### 7. `<picture>` 标签（响应式图片）

```html
<picture>
    <source media="(max-width: 768px)" srcset="small.jpg">
    <source media="(max-width: 1200px)" srcset="medium.jpg">
    <img src="large.jpg" alt="响应式图片">
</picture>
```

**作用：** 根据屏幕大小加载不同的图片

**何时使用：**
- 手机屏幕需要较小的图片（节省流量）
- 大屏幕需要高分辨率的图片

## 第三部分：路径讲解

这是很多初学者容易出错的地方。理解路径对网页开发至关重要。

### 1. 两种路径类型

#### 绝对路径

```html
<!-- 从网站根目录开始 -->
<a href="/about.html">关于</a>
<img src="/images/logo.png" alt="logo">

<!-- 完整的网址 -->
<a href="https://example.com/about.html">关于</a>
<img src="https://example.com/images/logo.png" alt="logo">
```

**特点：**
- 从网站的根目录开始
- 无论从哪个页面访问，路径都是一样的
- 以 `/` 开头表示网站根目录

**什么时候用：**
- 链接到网站的主要页面
- 大型网站推荐使用

#### 相对路径

```html
<!-- 同一文件夹 -->
<img src="logo.png" alt="logo">

<!-- 子文件夹 -->
<img src="images/logo.png" alt="logo">

<!-- 上级文件夹 -->
<img src="../logo.png" alt="logo">
```

**特点：**
- 相对于当前文件的位置
- 不以 `/` 开头
- 简洁易用

**什么时候用：**
- 项目开发时（大多数情况）
- 本地文件

---

### 2. 路径符号

```html
.     当前文件夹
..    上一级文件夹
/     分隔符（表示文件夹分隔）
```

**示例：**

```
网站结构：
website/
├─ index.html
├─ about.html
├─ images/
│  └─ logo.png
└─ css/
   └─ style.css
```

**从 about.html 中：**

```html
<!-- 链接到 index.html（同一级） -->
<a href="index.html">首页</a>

<!-- 链接到 images/logo.png（进入子文件夹） -->
<img src="images/logo.png" alt="logo">

<!-- 链接到 css/style.css（进入子文件夹） -->
<link rel="stylesheet" href="css/style.css">
```

**从 images/photo.html 中：**

```html
<!-- 回到上一级，然后链接到 index.html -->
<a href="../index.html">首页</a>

<!-- 回到上一级，然后进入 css 文件夹 -->
<link rel="stylesheet" href="../css/style.css">
```

---

### 3. 常见路径错误

```html
<!-- ❌ 错误：漏掉了文件扩展名 -->
<a href="about">关于</a>

<!-- ✅ 正确 -->
<a href="about.html">关于</a>

<!-- ❌ 错误：混用路径符号 -->
<img src="./images\logo.png">  <!-- Windows 反斜杠不对 -->

<!-- ✅ 正确：总是用正斜杠 -->
<img src="./images/logo.png">

<!-- ❌ 错误：路径不存在 -->
<img src="image/logo.png">  <!-- 实际是 images 文件夹 -->

<!-- ✅ 正确 -->
<img src="images/logo.png">
```

## 💡 综合最佳实践

### 好的链接示例

```html
<nav>
    <!-- 明确的链接文本 -->
    <a href="/about.html" title="了解我们的公司">关于我们</a>
    
    <!-- 外部链接在新标签打开 -->
    <a href="https://github.com/example" target="_blank" rel="nofollow">
        GitHub
    </a>
    
    <!-- 页面内跳转 -->
    <a href="#products">我们的产品</a>
</nav>
```

### 好的图片示例

```html
<!-- 本地图片 -->
<img src="images/product.jpg" 
     alt="高质量的产品照片：蓝色外壳，白色按钮"
     width="400">

<!-- 响应式图片 -->
<picture>
    <source srcset="images/logo-small.png" media="(max-width: 600px)">
    <source srcset="images/logo-medium.png" media="(max-width: 1200px)">
    <img src="images/logo-large.png" alt="公司 logo">
</picture>
```
