第 1 部分：链接 <a> 标签

什么是 <a> 标签？

<a> 标签（anchor，锚点）用来创建超链接。点击链接可以跳转到另一个页面或网站。

1. 最基础的外部链接

这是链接到外部网站的方式：

点击这里访问百度

代码：

<a href="https://www.baidu.com">点击这里访问百度</a>

说明： href 属性包含链接的 URL 地址

2. 页面内部链接（锚点跳转）

你可以点击下面的链接跳转到这个页面的其他部分：

  • 跳转到"链接部分"
  • 跳转到"图片部分"
  • 跳转到"路径讲解"

代码：

<a href="#images">跳转到图片部分</a>

<h2 id="images">图片部分</h2>

说明： # 表示页面内跳转，id 是目标元素的标识符

3. 在新标签页打开链接

在新标签页打开 Google

代码：

<a href="https://www.google.com" target="_blank">在新标签页打开 Google</a>

说明： target="_blank" 会在新标签页打开链接，不会关闭当前页面

4. 邮件链接

发送邮件给我

代码：

<a href="mailto:example@example.com">发送邮件给我</a>

说明： 点击链接会打开用户的默认邮箱应用

5. 电话链接

拨打我的电话

代码：

<a href="tel:+8613800138000">拨打我的电话</a>

说明： 在手机上点击会拨打电话，在电脑上通常没有反应

6. 链接的常用属性

href 属性（必需）

访问 GitHub

作用： 指定链接的目标地址

target 属性

  • target="_self" - 在当前窗口打开（默认）
  • target="_blank" - 在新标签页打开
  • target="_parent" - 在父框架打开
  • target="_top" - 在顶层窗口打开

title 属性（提示）

点击查看维基百科（鼠标悬停看提示）

作用： 鼠标悬停时显示提示文本

rel 属性

外部链接（告诉搜索引擎不追踪）

作用： 定义链接与当前文档的关系

7. 所有链接类型总结

链接类型 href 值 示例
外部网站 https://... <a href="https://google.com">
相对路径 page.html <a href="page.html">
页面内跳转 #id <a href="#top">
邮件 mailto: <a href="mailto:user@example.com">
电话 tel: <a href="tel:+86138...">

第 2 部分：图片 <img> 标签

什么是 <img> 标签？

<img> 标签用来在网页中显示图片。它是一个自闭合标签（没有结束标签）。

1. 最基础的图片

代码：

<img src="image.jpg" alt="图片描述">

说明： src 是图片路径，alt 是图片加载失败时显示的文本

2. 图片的属性

src 属性（必需）

作用： 指定图片的路径（本地文件或网络地址）

示例：

  • src="image.jpg" - 同一文件夹
  • src="images/photo.jpg" - images 文件夹
  • src="https://example.com/image.jpg" - 网络图片

alt 属性（必需）

作用： 当图片无法加载时显示的替代文本

为什么重要：

  • 如果图片加载失败，用户能看到文字说明
  • 屏幕阅读器会读出 alt 文本（帮助视障人士）
  • 提高网站的可访问性
  • 有助于搜索引擎理解图片内容

width 和 height 属性

作用： 指定图片的显示宽度和高度（单位：像素）

好处：

  • 控制图片在页面上的大小
  • 可以防止图片抖动（加载时）
  • 通常应该只指定宽度，让浏览器自动计算高度以保持比例

title 属性

作用： 鼠标悬停时显示提示文本

3. 常见的图片格式

格式 扩展名 用途 特点
JPEG .jpg, .jpeg 照片 有损压缩，文件小
PNG .png 图形、logo 无损，支持透明
GIF .gif 动画 支持动画，文件小
WebP .webp 现代格式 体积小，质量好
SVG .svg 矢量图 可缩放，不失清晰

4. 响应式图片（高级）

方法 1：使用 max-width

<img src="image.jpg" alt="描述" style="max-width: 100%;">

说明： 图片宽度不会超过容器，自动缩放以适应屏幕大小

方法 2：使用 srcset 属性

<img src="small.jpg"
    srcset="small.jpg 800w, medium.jpg 1200w, large.jpg 1600w"
    alt="描述">

说明： 根据屏幕大小加载不同尺寸的图片

第 3 部分：路径讲解（重要）

什么是路径？

路径是告诉浏览器文件在哪里的方式。有两种主要的路径类型：

1. 绝对路径

定义： 从网站根目录开始的完整路径

示例：

  • /images/photo.jpg - 网站根目录的 images 文件夹
  • /about.html - 网站根目录的 about.html 文件
  • https://example.com/images/photo.jpg - 完整的网址

什么时候用： 链接到同一网站内的文件

2. 相对路径

定义： 相对于当前文件位置的路径

示例：

  • image.jpg - 当前文件夹内的图片
  • images/photo.jpg - 当前文件夹的 images 子文件夹
  • ../images/photo.jpg - 上一级文件夹的 images 文件夹
  • ../../images/photo.jpg - 上两级文件夹的 images 文件夹

什么时候用： 大多数时候，特别是本地文件

3. 路径结构示例

my-website/
│
├─ index.html (主页)
├─ about.html
│
├─ images/
│  ├─ logo.png
│  ├─ photo1.jpg
│  └─ photo2.jpg
│
├─ css/
│  └─ style.css
│
└─ js/
   └─ script.js

4. 从 about.html 中的路径示例

链接到 index.html：

  • <a href="index.html"> (相对)
  • <a href="/index.html"> (绝对)

链接到 images/logo.png：

  • <img src="images/logo.png"> (相对)
  • <img src="/images/logo.png"> (绝对)

链接到 css/style.css：

  • <link rel="stylesheet" href="css/style.css"> (相对)

5. 特殊路径

符号 含义 示例
./ 当前文件夹 ./image.jpg
../ 上一级文件夹 ../images/photo.jpg
/ 网站根目录 /images/photo.jpg
