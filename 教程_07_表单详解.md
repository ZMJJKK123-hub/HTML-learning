## 第 1 部分：表单基础

### 什么是表单？

**表单**（Form）是用户输入数据的区域，由各种**输入控件**组成，最终将数据发送到服务器处理。

```
┌─────────────────────────┐
│  登录                   │
│  ┌───────────────────┐ │
│  │ 用户名            │ │  ← 输入框
│  └───────────────────┘ │
│  ┌───────────────────┐ │
│  │ ●●●●●●            │ │  ← 密码框
│  └───────────────────┘ │
│  ☐ 记住我             │  ← 复选框
│  [登录] [取消]        │  ← 按钮
└─────────────────────────┘
```

### 基本语法

```html
<form action="/submit" method="post">
    <!-- 表单内容 -->
</form>
```

**标签说明：**
- `<form>` - 表单容器
- `action` - 数据提交的目标地址（服务器端）
- `method` - 提交方式：`get` 或 `post`

### 代码详解

```html
<form action="/submit" method="post">
```

**`action` 属性：**
- 指定表单数据提交到哪里
- 例如：`action="/login"` 表示提交到 /login 地址
- 如果不写，默认提交到当前页面

**`method` 属性：**
- `get` - 数据显示在 URL 中（不安全，用于搜索）
- `post` - 数据隐藏在请求中（安全，用于登录、注册）

**示例对比：**

```html
<!-- GET 方式：数据显示在 URL -->
<form action="/search" method="get">
    <input type="text" name="q">
    <button type="submit">搜索</button>
</form>
<!-- 提交后 URL：/search?q=用户输入的内容 -->

<!-- POST 方式：数据隐藏 -->
<form action="/login" method="post">
    <input type="text" name="username">
    <input type="password" name="password">
    <button type="submit">登录</button>
</form>
<!-- 提交后 URL：/login（数据不显示） -->
```

### 最简单的表单

```html
<form>
    <label for="username">用户名：</label>
    <input type="text" id="username" name="username">
    <button type="submit">提交</button>
</form>
```

**结构说明：**
1. `<form>` - 表单容器
2. `<label>` - 标签（描述输入框的用途）
3. `<input>` - 输入框
4. `<button>` - 按钮

## 第 2 部分：文本输入类型

### 1. 普通文本框 (text)

**用途：**输入任何文本（姓名、地址等）

```html
<label for="name">姓名：</label>
<input type="text" id="name" name="name" placeholder="请输入你的姓名">
```

**属性说明：**
- `type="text"` - 文本类型（默认）
- `id="name"` - 唯一标识符，与 label 的 for 对应
- `name="name"` - 提交时的字段名
- `placeholder` - 占位符提示

### 2. 密码框 (password)

**用途：**输入密码（自动隐藏字符）

```html
<label for="password">密码：</label>
<input type="password" id="password" name="password" placeholder="请输入密码">
```

**特点：**
- 输入的字符显示为 ●●●●●
- 无法复制粘贴（某些浏览器）
- 安全性更高

### 3. 邮箱 (email)

**用途：**输入邮箱地址

```html
<label for="email">邮箱：</label>
<input type="email" id="email" name="email" placeholder="your@email.com">
```

**好处：**
- **自动验证**邮箱格式（必须有 @ 符号）
- 手机上显示**邮箱专用键盘**（有 @ 键）
- 提交前会检查格式

**验证示例：**
```
user@example.com    ✅ 有效
user@example        ❌ 无效（缺少 .com）
user                ❌ 无效（缺少 @）
```

### 4. 数字 (number)

**用途：**输入数字

```html
<label for="age">年龄：</label>
<input type="number" id="age" name="age" min="1" max="120" step="1">
```

**属性：**
- `min` - 最小值
- `max` - 最大值
- `step` - 步长（每次增减的数量）

**示例：**
```html
<!-- 年龄：1-120 -->
<input type="number" min="1" max="120">

<!-- 价格：允许小数，步长 0.01 -->
<input type="number" min="0" step="0.01">
```

### 5. 电话 (tel)

**用途：**输入电话号码

```html
<label for="phone">电话：</label>
<input type="tel" id="phone" name="phone" placeholder="138-0000-0000">
```

**特点：**
- 手机上显示**数字键盘**
- 不会自动验证格式（因为各国格式不同）
- 可以配合 `pattern` 属性验证

### 6. 网址 (url)

**用途：**输入网址

```html
<label for="website">网站：</label>
<input type="url" id="website" name="website" placeholder="https://example.com">
```

**验证：**
```
https://example.com    ✅ 有效
http://example.com     ✅ 有效
example.com            ❌ 无效（缺少协议）
```

### 7. 搜索框 (search)

**用途：**搜索输入

```html
<label for="search">搜索：</label>
<input type="search" id="search" name="search" placeholder="输入关键词...">
```

**特点：**
- 外观可能略有不同（圆角）
- 输入内容后会显示清除按钮（×）

## 第 3 部分：日期和时间输入

### 1. 日期 (date)

```html
<label for="birthday">生日：</label>
<input type="date" id="birthday" name="birthday">
```

**效果：**浏览器会显示日期选择器（日历）

### 2. 时间 (time)

```html
<label for="appointment">预约时间：</label>
<input type="time" id="appointment" name="appointment">
```

**格式：**24 小时制，例如 14:30

### 3. 日期时间 (datetime-local)

```html
<label for="meeting">会议时间：</label>
<input type="datetime-local" id="meeting" name="meeting">
```

**包含：**日期 + 时间

### 4. 月份 (month)

```html
<label for="month">选择月份：</label>
<input type="month" id="month" name="month">
```

**格式：**2024-12

### 5. 星期 (week)

```html
<label for="week">选择星期：</label>
<input type="week" id="week" name="week">
```

**格式：**2024-W52（2024 年第 52 周）

## 第 4 部分：选择输入

### 1. 单选按钮 (radio)

**用途：**从多个选项中选择**一个**

```html
<fieldset>
    <legend>性别</legend>
    <label>
        <input type="radio" name="gender" value="male" checked>
        男
    </label>
    <label>
        <input type="radio" name="gender" value="female">
        女
    </label>
    <label>
        <input type="radio" name="gender" value="other">
        其他
    </label>
</fieldset>
```

**重要规则：**
- 同一组单选按钮必须有**相同的 name**
- 只能选择一个
- `checked` 属性表示默认选中
- `value` 是提交时的值

**错误示例：**
```html
<!-- ❌ 错误：name 不同，可以同时选择多个 -->
<input type="radio" name="gender1" value="male"> 男
<input type="radio" name="gender2" value="female"> 女

<!-- ✅ 正确：name 相同，只能选一个 -->
<input type="radio" name="gender" value="male"> 男
<input type="radio" name="gender" value="female"> 女
```

### 2. 复选框 (checkbox)

**用途：**从多个选项中选择**多个**

```html
<fieldset>
    <legend>兴趣爱好</legend>
    <label>
        <input type="checkbox" name="hobbies" value="reading">
        阅读
    </label>
    <label>
        <input type="checkbox" name="hobbies" value="sports">
        运动
    </label>
    <label>
        <input type="checkbox" name="hobbies" value="music">
        音乐
    </label>
</fieldset>
```

**特点：**
- 可以选择多个
- `checked` 表示默认选中
- 常用于"同意条款"、"记住我"等

**示例：同意条款**
```html
<label>
    <input type="checkbox" name="agree" required>
    我同意<a href="#">服务条款</a>
</label>
```

### 3. 下拉选择框 (select)

**用途：**从下拉列表中选择一项

```html
<label for="city">选择城市：</label>
<select id="city" name="city">
    <option value="">--请选择--</option>
    <option value="beijing">北京</option>
    <option value="shanghai">上海</option>
    <option value="guangzhou">广州</option>
    <option value="shenzhen">深圳</option>
</select>
```

**结构说明：**
- `<select>` - 下拉框容器
- `<option>` - 每个选项
- `value` - 提交时的值
- 标签内容 - 显示给用户的文字

**默认选中：**
```html
<select name="city">
    <option value="">--请选择--</option>
    <option value="beijing" selected>北京</option>
    <option value="shanghai">上海</option>
</select>
```

### 4. 多选下拉框

```html
<label for="languages">选择语言（按住 Ctrl 多选）：</label>
<select id="languages" name="languages" multiple size="5">
    <option value="html">HTML</option>
    <option value="css">CSS</option>
    <option value="javascript">JavaScript</option>
    <option value="python">Python</option>
    <option value="java">Java</option>
</select>
```

**属性：**
- `multiple` - 允许多选
- `size` - 显示多少行

### 5. 分组选择框 (optgroup)

```html
<label for="country">选择国家：</label>
<select id="country" name="country">
    <optgroup label="亚洲">
        <option value="china">中国</option>
        <option value="japan">日本</option>
        <option value="korea">韩国</option>
    </optgroup>
    <optgroup label="欧洲">
        <option value="uk">英国</option>
        <option value="france">法国</option>
        <option value="germany">德国</option>
    </optgroup>
</select>
```

**用途：**把相关选项分组，更易查找

## 第 5 部分：文本域和其他输入

### 1. 文本域 (textarea)

**用途：**输入多行文本（留言、评论等）

```html
<label for="message">留言：</label>
<textarea id="message" name="message" rows="5" cols="50" placeholder="请输入你的留言..."></textarea>
```

**属性：**
- `rows` - 可见行数
- `cols` - 可见列数（宽度）
- 也可以用 CSS 控制大小

**与 input 的区别：**
```html
<!-- input：单行 -->
<input type="text" name="title">

<!-- textarea：多行 -->
<textarea name="content"></textarea>
```

### 2. 文件上传 (file)

**用途：**上传文件

```html
<!-- 单个文件 -->
<label for="avatar">上传头像：</label>
<input type="file" id="avatar" name="avatar">

<!-- 多个文件 -->
<label for="documents">上传文档：</label>
<input type="file" id="documents" name="documents" multiple>

<!-- 只接受图片 -->
<label for="photo">上传照片：</label>
<input type="file" id="photo" name="photo" accept="image/*">
```

**属性：**
- `multiple` - 允许选择多个文件
- `accept` - 限制文件类型

**accept 示例：**
```html
<!-- 只接受图片 -->
<input type="file" accept="image/*">

<!-- 只接受 PDF -->
<input type="file" accept=".pdf">

<!-- 只接受图片和 PDF -->
<input type="file" accept="image/*,.pdf">

<!-- 只接受特定图片格式 -->
<input type="file" accept="image/png,image/jpeg">
```

### 3. 颜色选择器 (color)

```html
<label for="favorite-color">选择颜色：</label>
<input type="color" id="favorite-color" name="color" value="#667eea">
```

**效果：**浏览器会显示颜色选择器

### 4. 范围滑块 (range)

```html
<label for="volume">音量：</label>
<input type="range" id="volume" name="volume" min="0" max="100" value="50">
```

**属性：**
- `min` - 最小值
- `max` - 最大值
- `value` - 默认值
- `step` - 步长

### 5. 隐藏字段 (hidden)

**用途：**存储不需要用户看到的数据

```html
<input type="hidden" name="user_id" value="12345">
<input type="hidden" name="token" value="abc123xyz">
```

**应用场景：**
- 用户 ID
- 安全令牌
- 表单版本号
- 任何需要提交但不需要显示的数据

## 第 6 部分：按钮类型

### 1. 提交按钮 (submit)

**用途：**提交表单

```html
<!-- 方式 1：button 标签 -->
<button type="submit">提交表单</button>

<!-- 方式 2：input 标签 -->
<input type="submit" value="提交表单">
```

**区别：**
- `<button>` 可以包含 HTML（图标、图片等）
- `<input>` 只能显示纯文本

**示例：带图标的按钮**
```html
<button type="submit">
    ✓ 提交表单
</button>
```

### 2. 重置按钮 (reset)

**用途：**清空表单，恢复初始值

```html
<button type="reset">重置表单</button>
<input type="reset" value="重置表单">
```

**⚠️ 注意：**用户可能误点重置按钮，导致辛苦填写的内容丢失，所以要慎用！

### 3. 普通按钮 (button)

**用途：**执行 JavaScript 代码

```html
<button type="button" onclick="alert('Hello!')">点击我</button>
<input type="button" value="点击我" onclick="alert('Hello!')">
```

**与 submit 的区别：**
- `type="button"` - 不会提交表单
- `type="submit"` - 会提交表单

## 第 7 部分：重要属性

### 1. required - 必填项

**用途：**标记字段为必填

```html
<label for="username">
    用户名 <span style="color: red;">*</span>
</label>
<input type="text" id="username" name="username" required>
```

**效果：**如果不填写，提交时浏览器会提示错误

### 2. placeholder - 占位符

**用途：**在输入框内显示提示文字

```html
<input type="text" placeholder="请输入你的姓名">
<input type="email" placeholder="example@email.com">
```

**特点：**
- 用户开始输入时，占位符会消失
- 不是实际的值（不会被提交）

### 3. maxlength - 最大长度

**用途：**限制最多输入多少字符

```html
<label for="code">验证码（最多 6 位）：</label>
<input type="text" id="code" maxlength="6">
```

**效果：**超过 6 个字符就无法继续输入

### 4. minlength - 最小长度

**用途：**限制最少输入多少字符

```html
<label for="password">密码（至少 8 位）：</label>
<input type="password" id="password" minlength="8">
```

**效果：**少于 8 个字符时，提交会报错

### 5. pattern - 正则表达式验证

**用途：**使用正则表达式验证格式

```html
<!-- 手机号：11 位数字 -->
<input type="text" pattern="[0-9]{11}" placeholder="13800138000">

<!-- 邮政编码：6 位数字 -->
<input type="text" pattern="[0-9]{6}" placeholder="100000">

<!-- 用户名：3-16 位字母或数字 -->
<input type="text" pattern="[A-Za-z0-9]{3,16}">
```

**常用正则：**
- `[0-9]{11}` - 11 位数字
- `[A-Za-z]+` - 只能是字母
- `[0-9]+` - 只能是数字
- `[A-Za-z0-9]+` - 字母或数字

### 6. disabled - 禁用

**用途：**禁用输入框（无法编辑，无法提交）

```html
<input type="text" value="已禁用" disabled>
<button type="submit" disabled>提交（禁用）</button>
```

**特点：**
- 灰色显示
- 无法聚焦
- **不会**被提交到服务器

### 7. readonly - 只读

**用途：**只读（无法编辑，但会提交）

```html
<input type="text" value="只读内容" readonly>
```

**与 disabled 的区别：**

| 特性 | disabled | readonly |
|------|----------|----------|
| 可以编辑 | ❌ | ❌ |
| 可以选中文本 | ❌ | ✅ |
| 会被提交 | ❌ | ✅ |
| 外观 | 灰色 | 正常 |

### 8. autocomplete - 自动完成

**用途：**控制浏览器是否记住输入

```html
<!-- 开启自动完成 -->
<input type="text" name="name" autocomplete="on">

<!-- 关闭自动完成 -->
<input type="text" name="secret" autocomplete="off">
```

**常用值：**
- `on` - 开启
- `off` - 关闭
- `name` - 姓名
- `email` - 邮箱
- `tel` - 电话
- `address-line1` - 地址第 1 行
- 等等...

## 第 8 部分：实际应用示例

### 示例 1：登录表单

```html
<form action="/login" method="post">
    <h2>用户登录</h2>
    
    <div>
        <label for="login-email">
            邮箱 <span style="color: red;">*</span>
        </label>
        <input type="email" 
               id="login-email" 
               name="email" 
               required 
               placeholder="your@email.com">
    </div>
    
    <div>
        <label for="login-password">
            密码 <span style="color: red;">*</span>
        </label>
        <input type="password"

               name="password" 
               required 
               minlength="8"
               placeholder="至少 8 位">
    </div>

        </label>
    </div>

</form>
```

    <div>
        <label for="contact-message">消息 *</label>
        <textarea id="contact-message" name="message" rows="5" required></textarea>
    </div>
    
    <button type="submit">发送</button>
    <button type="reset">清空</button>

## 第 9 部分：最佳实践

### ✅ 好的做法

#### 1. 始终使用 &lt;label&gt; 标签

```html
<!-- ✅ 好：使用 label -->
<label for="username">用户名：</label>
<input type="text" id="username" name="username">

<!-- ❌ 差：没有 label -->
用户名：<input type="text" name="username">
```

**好处：**
- 点击标签也能聚焦输入框
- 屏幕阅读器能理解
- 提高可访问性

#### 2. 使用合适的输入类型

```html
<!-- ✅ 好：使用 email 类型 -->
<input type="email" name="email">

<!-- ❌ 差：用 text 类型 -->
<input type="text" name="email">
```

**好处：**
- 自动验证格式
- 手机上显示专用键盘
- 更好的用户体验

#### 3. 提供清晰的提示

```html
<!-- ✅ 好：有 placeholder 和提示 -->
<label for="password">密码：</label>
<input type="password" 
       id="password" 
       placeholder="至少 8 位" 
       minlength="8">
<small>密码必须包含字母和数字</small>
```

#### 4. 标记必填项

```html
<label for="email">
    邮箱 <span style="color: red;">*</span>
</label>
<input type="email" id="email" required>
```

#### 5. 分组相关字段

```html
<fieldset>
    <legend>个人信息</legend>
    <!-- 相关字段 -->
</fieldset>

<fieldset>
    <legend>联系方式</legend>
    <!-- 相关字段 -->
</fieldset>
```

### ❌ 避免的做法

#### 1. 不要使用过多的必填项

```
❌ 错误：要求用户填写 20 个必填项
✅ 正确：只要求必要的信息，其他设为可选
```

#### 2. 不要使用模糊的提示

```
❌ 错误："请输入正确的格式"（什么格式？）
✅ 正确："请输入 11 位手机号，例如：13800138000"
```

#### 3. 不要禁用复制粘贴

```
❌ 错误：在密码确认框禁用粘贴
✅ 正确：允许用户使用密码管理器
```

#### 4. 不要使用太多下拉选择

```
❌ 错误：让用户从 100 个国家中选择
✅ 正确：提供搜索功能，或者只列出常用的
```

## 常见问题（FAQ）

### Q1: name 和 id 有什么区别？

**A:**
- `name` - 提交到服务器的字段名（必须）
- `id` - 页面内的唯一标识，用于 label 的 for 属性

```html
<label for="username">用户名：</label>
<input type="text" id="username" name="username">
```

### Q2: GET 和 POST 有什么区别？

**A:**

| 特性 | GET | POST |
|------|-----|------|
| 数据位置 | URL 中（可见） | 请求体中（不可见） |
| 安全性 | 低 | 高 |
| 长度限制 | 有（约 2KB） | 无限制 |
| 用途 | 搜索、筛选 | 登录、注册、提交 |
| 可收藏 | 可以 | 不可以 |

### Q3: 如何让两个输入框在同一行？

**A:** 使用 CSS 的 flexbox：

```html
<div style="display: flex; gap: 1rem;">
    <div style="flex: 1;">
        <label>名</label>
        <input type="text">
    </div>
    <div style="flex: 1;">
        <label>姓</label>
        <input type="text">
    </div>
</div>
```

### Q4: 如何验证两个密码框是否一致？

**A:** 使用 JavaScript：

```html
<form onsubmit="return checkPassword()">
    <input type="password" id="password" name="password">
    <input type="password" id="confirm" name="confirm">
    <button type="submit">提交</button>
</form>

<script>
function checkPassword() {
    const password = document.getElementById('password').value;
    const confirm = document.getElementById('confirm').value;
    
    if (password !== confirm) {
        alert('两次密码不一致！');
        return false;  // 阻止提交
    }
    return true;  // 允许提交
}
</script>
```

### Q5: 如何美化表单样式？

**A:** 使用 CSS：

```css
/* 输入框样式 */
input[type="text"],
input[type="email"],
input[type="password"] {
    width: 100%;
    padding: 10px;
    border: 1px solid #ddd;
    border-radius: 4px;
    font-size: 1rem;
}

/* 聚焦时的样式 */
input:focus {
    outline: none;
    border-color: #667eea;
    box-shadow: 0 0 0 3px rgba(102, 126, 234, 0.1);
}

/* 按钮样式 */
button {
    padding: 10px 20px;
    background-color: #4CAF50;
    color: white;
    border: none;
    border-radius: 4px;
    cursor: pointer;
}

button:hover {
    background-color: #45a049;
}
```
