# HTML 快速参考指南 (HTML Cheat Sheet)

这个文档列出了你已经学习过的所有 HTML 标签和属性。可以当作字典来使用。

---

## 📋 文档结构标签

### 基础标签

| 标签 | 属性 | 用途 | 示例 |
|------|------|------|------|
| `<!DOCTYPE html>` | - | 文档声明 | `<!DOCTYPE html>` |
| `<html>` | `lang` | 根元素 | `<html lang="zh-CN">` |
| `<head>` | - | 页面配置 | `<head>...</head>` |
| `<body>` | - | 页面内容 | `<body>...</body>` |
| `<title>` | - | 页面标题 | `<title>我的网页</title>` |
| `<meta>` | `charset`, `name`, `content` | 元数据 | `<meta charset="UTF-8">` |

### 常用 Meta 标签

```html
<!-- 字符编码 -->
<meta charset="UTF-8">

<!-- 视口设置（响应式设计） -->
<meta name="viewport" content="width=device-width, initial-scale=1.0">

<!-- 页面描述（SEO） -->
<meta name="description" content="页面描述">

<!-- 页面作者 -->
<meta name="author" content="作者名称">

<!-- 网页关键词 -->
<meta name="keywords" content="关键词1, 关键词2">

<!-- 刷新页面 -->
<meta http-equiv="refresh" content="30">
```

## 🔗 链接相关

### 链接 `<a>` 标签

```html
<!-- 基础属性 -->
<a href="url">链接文本</a>

<!-- 完整属性 -->
<a href="url" target="_blank" title="提示">链接</a>

<!-- href 的各种值 -->
<a href="https://example.com">外部网站</a>
<a href="about.html">网站内页面</a>
<a href="#section">页面内跳转</a>
<a href="mailto:email@example.com">邮件</a>
<a href="tel:+8613800138000">电话</a>
<a href="/files/doc.pdf" download>下载</a>

<!-- target 属性值 -->
<a href="url" target="_self">当前窗口（默认）</a>
<a href="url" target="_blank">新标签页</a>
<a href="url" target="_parent">父框架</a>
<a href="url" target="_top">顶层窗口</a>

<!-- rel 属性 -->
<a href="url" rel="nofollow">不追踪链接</a>
<a href="url" rel="next">下一页</a>
<a href="url" rel="prev">上一页</a>
```

### 锚点定位

```html
<!-- 链接部分 -->
<a href="#section1">跳转到第一部分</a>

<!-- 目标部分 -->
<h2 id="section1">第一部分</h2>

<!-- 完整 URL 中也可以用锚点 -->
<a href="page.html#section1">外部页面的某个部分</a>
```

## 🖼️ 图片相关

### 图片 `<img>` 标签

```html
<!-- 基础 -->
<img src="image.jpg" alt="描述">

<!-- 完整属性 -->
<img src="image.jpg" 
     alt="图片描述"
     width="300" 
     height="200"
     title="鼠标悬停提示">

<!-- 本地图片 -->
<img src="image.jpg">           <!-- 同文件夹 -->
<img src="images/photo.jpg">    <!-- 子文件夹 -->
<img src="../images/photo.jpg"> <!-- 上级文件夹 -->

<!-- 网络图片 -->
<img src="https://example.com/image.jpg">

<!-- 响应式 -->
<img src="image.jpg" style="max-width: 100%;">
```

### 响应式图片 `<picture>` 标签

```html
<picture>
    <source media="(max-width: 600px)" srcset="small.jpg">
    <source media="(max-width: 1200px)" srcset="medium.jpg">
    <img src="large.jpg" alt="响应式图片">
</picture>
```

| `<ul>` | 无序列表 | • |

<ul>
    <li>前端

        </ul>
    </li>
    <li>后端</li>

    <dt>HTML</dt>
    <dd>超文本标记语言</dd>

    <tr>

## 📐 路径速查

### 相对路径（推荐用于本地）

```html
<!-- 同一文件夹 -->
<img src="logo.png">

<!-- 子文件夹 -->
<img src="images/logo.png">

<!-- 多个子文件夹 -->
<img src="assets/images/logo.png">

<!-- 上一级文件夹 -->
<img src="../logo.png">

<!-- 上一级的子文件夹 -->
<img src="../images/logo.png">

<!-- 上两级文件夹 -->
<img src="../../logo.png">
```

### 绝对路径（推荐用于网站链接）

```html
<!-- 从网站根目录开始 -->
<img src="/images/logo.png">
<a href="/about.html">关于</a>

<!-- 完整 URL -->
<img src="https://example.com/logo.png">
<a href="https://example.com/about.html">关于</a>
```

## 💡 最佳实践速查

### ✅ 做这些

```html
<!-- 1. 始终包含 DOCTYPE -->
<!DOCTYPE html>

<!-- 2. 始终使用正确的语言标签 -->
<html lang="zh-CN">

<!-- 3. 为所有图片提供 alt 文本 -->
<img src="photo.jpg" alt="描述">

<!-- 4. 为表单字段提供标签 -->
<label for="email">邮箱：</label>
<input id="email" type="email">

<!-- 5. 使用语义化标签 -->
<header>...</header>
<nav>...</nav>
<main>...</main>
<footer>...</footer>

<!-- 6. 正确嵌套标签 -->
<div>
    <p>内容</p>
</div>

<!-- 7. 为页面内链接使用锚点 -->
<a href="#section">跳转</a>
<section id="section">...</section>
```

### ❌ 避免这些

```html
<!-- 1. 不要省略 DOCTYPE -->
❌ <head>...</head>  <!-- 没有 <!DOCTYPE> -->

<!-- 2. 不要跳过标题等级 -->
❌ <h1>标题</h1>
   <h3>副标题</h3>  <!-- 应该是 <h2> -->

<!-- 3. 不要省略 alt 文本 -->
❌ <img src="photo.jpg">

<!-- 4. 不要用 <div> 替代语义标签 -->
❌ <div id="header">标题</div>
✅ <header>标题</header>

<!-- 5. 不要错误地嵌套标签 -->
❌ <p><div>内容</div></p>
✅ <div><p>内容</p></div>

<!-- 6. 不要过度使用 <br> 换行 -->
❌ 段落1<br><br><br>段落2
✅ <p>段落1</p><p>段落2</p>

<!-- 7. 不要在链接中写 "点击这里" -->
❌ <a href="about.html">点击这里</a> 了解我们
✅ <a href="about.html">了解我们</a>
```
