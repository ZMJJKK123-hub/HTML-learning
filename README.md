## 📚 项目结构

这个项目包含 **HTML 基础 1-7 步、CSS 进阶 8-14 步、JS 进阶 15-20 步（已完成）**，每个步骤都有：

1. **实践文件** - 在浏览器中看到实际效果
2. **详细教程** - 理解每个概念的原理
3. **学习指南** - 跟踪你的学习进度

### 学习路线图

```
HTML 基础
第 1 步：最基础的 HTML
└─ DOCTYPE, html, head, body
   ↓
第 2 步：页面结构与语义化
└─ header, nav, main, section, footer
   ↓
第 3 步：文本格式化
└─ 标题、加粗、斜体、代码等
   ↓
第 4 步：链接和图片
└─ 创建链接、插入图片、理解路径
   ↓
第 5 步：列表
└─ ul/ol/描述列表，嵌套与样式
   ↓
第 6 步：表格
└─ thead/tbody/tfoot、合并单元格、样式与可访问性
   ↓
第 7 步：表单
└─ input/select/textarea、验证与可访问性

CSS 进阶
第 8 步：CSS 基础与选择器
└─ 内联/内部/外部、选择器与优先级
   ↓
第 9 步：排版与盒模型进阶
└─ 字体体系、display 差异、BFC、阴影与对齐
   ↓
第 10 步：布局与 Flexbox
└─ position、float/clearfix、Flex 容器与项目
   ↓
第 11 步：Grid 与响应式
└─ fr/minmax/auto-fit、grid-area、断点策略
   ↓
第 12 步：过渡与动效
└─ transition、transform、keyframes、性能优化
   ↓
第 13 步：变量与层叠工程化
└─ tokens、CSS 变量、@layer、特异性管理
   ↓
第 14 步：综合实战小站
└─ 主题切换、Grid+Flex、表单交互、动效与可访问性

JavaScript（进行中）
第 15 步：JS 基础与语法
└─ 变量/作用域、条件/循环、函数、对象/数组、模块初体验
   ↓
第 16 步：DOM 与事件
└─ 选择/修改 DOM、事件模型、委托、节流/防抖、基础表单校验
   ↓
第 17 步：异步与数据交互
└─ Promise/async-await、错误处理、并发合并、重试
   ↓
第 18 步：组件化与状态
└─ 状态管理模式、可复用组件（Tabs/Modal/Toast）、可访问性焦点管理
   ↓
第 19 步：性能与工程化
└─ 代码分割/动态导入、空闲调度、缓存与版本、Lint/Format/TS 入门（概念）
   ↓
第 20 步：综合项目
└─ 将 JS + CSS + 设计令牌整合成可部署的小型站点，含接口模拟/部署
```

## 💡 核心概念速览

### HTML 是什么？

**HTML** = HyperText Markup Language
- 用标签标记内容的含义
- 定义网页的结构
- 所有网站的基础

### 为什么要学 HTML？

1. **网页开发的基础**
   - CSS 和 JavaScript 都建立在 HTML 之上

2. **搜索引擎优化**
   - 良好的 HTML 结构帮助 Google、百度理解你的网页

3. **可访问性**
   - 视障人士使用屏幕阅读器需要好的 HTML

4. **跨平台兼容性**
   - 正确的 HTML 在任何设备上都能工作

## 📞 常见问题

### Q: 我完全不懂编程，能学会吗？
**A:** 完全可以！这个项目就是为初学者设计的。HTML 非常容易学。

### Q: 需要多长时间？
**A:** HTML 基础部分（1-4 步）需要 3-5 天；继续到 HTML 5-7 与 CSS 8-14 视你的节奏安排。

### Q: 我应该按顺序学吗？
**A:** 是的。每一步都建立在前一步的基础上。

### Q: 可以跳过某些部分吗？
**A:** 不建议。即使你觉得某个部分简单，也要完整学习。

### Q: 学 HTML 还需要学 CSS 和 JavaScript 吗？
**A:** 是的。HTML + CSS + JavaScript 才是完整的网页开发。但先把 HTML 学好最重要。

### Q: 这些教程会定期更新吗？
**A:** 会的。当学到新步骤时会添加新内容。
