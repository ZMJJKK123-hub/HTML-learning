# 运行与预览指南

适用：在本地查看 HTML/CSS/JS 演示与综合项目，进行基础调试与性能观察。

## 本地预览方式
- 直接双击 HTML：适合纯静态页面
- VS Code 扩展 Live Server：右键 HTML → Open with Live Server（自动热刷新）
- 静态服务（Node）：如 `npx serve` 或 `pnpm dlx serve` 启动根目录静态站点

## 网络与缓存调试
- DevTools Network：
  - 勾选 Disable cache（功能调试不要命中缓存）
  - 使用 Throttling 测试低网速体验（Fast 3G / Slow 3G）
- Performance/Lighthouse：记录关键指标（FCP/LCP/CLS/TTI/TTFB）

## 常用调试技巧
- Elements：检查 DOM 与样式层叠；查看可访问性（ARIA、对比度）
- Console：打印调试信息；捕捉错误堆栈
- Sources：打断点、单步调试；观察异步调用链
- Application：查看 LocalStorage/SessionStorage/IndexedDB；Service Worker 与缓存

## 生产预览建议
- 构建产物（如使用 Vite）：在 dist 下用静态服务或容器启动
- 验证缓存/指纹：检查响应头与资源文件名（hash）
- 断网测试：保证离线兜底页或关键静态资源可加载（如已启用 Service Worker）
