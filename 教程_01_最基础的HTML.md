# 第 1 步：最基础的 HTML 文件详解

## 📌 文件说明
这是一个最简单的 HTML 网页。打开 `01_hello_world.html` 就能在浏览器中看到页面。

---

## 🔍 逐行讲解

### 第 1 行：`<!DOCTYPE html>`
```html
<!DOCTYPE html>
```

**这是什么？**
- DOCTYPE 是"文档类型声明"的缩写
- **必须是文件的第一行**，不能有任何内容在它前面

**有什么用？**
- 告诉浏览器："嘿，我是一个现代的 HTML5 文档，请按照 HTML5 标准来解析我"
- 如果没有它，浏览器可能会用很久以前的方式解析页面，导致显示不正确

**记住这一点：**
> 每个 HTML 文件都必须以 `<!DOCTYPE html>` 开头

---

### 第 2 行：`<html lang="zh-CN">`
```html
<html lang="zh-CN">
```

**这是什么？**
- `<html>` 标签是整个网页的**根元素**，包含所有其他内容
- `lang="zh-CN"` 是一个**属性**，表示这个网页的语言是简体中文

**属性是什么？**
- 属性是在标签内部添加的额外信息
- 格式：`<标签名 属性名="属性值">`
- 例如：`lang="zh-CN"` 表示"语言属性，值为中文"

**其他常见的 lang 值：**
- `lang="en"` - 英文
- `lang="ja"` - 日文
- `lang="ko"` - 韩文

**记住这一点：**
> `<html>` 标签包裹整个网页，是最大的容器

---

### 第 3-5 行：`<head>` 部分
```html
<head>
    <meta charset="UTF-8">
    <title>我的第一个网页</title>
</head>
```

**`<head>` 是什么？**
- 头部，包含**网页的元信息**（即不在页面上显示的内容）
- 浏览器用这些信息来**配置和渲染**页面

**里面有什么？**

#### `<meta charset="UTF-8">`
- **作用**：定义字符编码方式
- **UTF-8**：一种国际通用的编码，支持所有语言符号（包括中文）
- **为什么重要**：没有它，中文会显示成乱码 🤪

#### `<title>我的第一个网页</title>`
- **作用**：定义浏览器标签页上显示的标题
- **打开网页后**：你会在浏览器的标签页上看到"我的第一个网页"这个文字
- **用处**：告诉用户这个页面是关于什么的

**实际例子：**
```
当你打开这个网页时，浏览器标签页会显示：
┌─────────────────────────────────────┐
│ 我的第一个网页  ✕  │ 其他标签页...  │
└─────────────────────────────────────┘
↑ 这就是 <title> 的内容
```

**记住这一点：**
> `<head>` 里的内容**不会显示在页面上**，它用来配置页面

---

### 第 6-8 行：`<body>` 部分
```html
<body>
    <h1>你好，世界！</h1>
    <p>这是我的第一个网页。</p>
</body>
```

**`<body>` 是什么？**
- 身体，包含**所有用户能看到的内容**
- 与 `<head>` 不同，这里的东西都会在网页上显示

**里面有什么？**

#### `<h1>你好，世界！</h1>`
- **h1**：最大的标题（1 级标题）
- **有 6 个标题等级**：`<h1>` 到 `<h6>`
- **规则**：一个页面通常只有一个 `<h1>`

#### `<p>这是我的第一个网页。</p>`
- **p**："paragraph"（段落）的缩写
- **作用**：包裹一段文字，段落前后会自动添加空白

**记住这一点：**
> `<body>` 里的所有内容都会显示在网页上

---

## 📊 网页的整体结构图

```
整个网页（<html>）
│
├─ 头部（<head>）
│  ├─ 字符编码
│  ├─ 标题
│  ├─ 样式表（之后会学到）
│  └─ 脚本文件（之后会学到）
│
└─ 身体（<body>）
   ├─ 标题（<h1>）
   └─ 段落（<p>）
```

---

## 💡 关键概念总结

| 概念 | 说明 | 例子 |
|------|------|------|
| **标签** | HTML 的基本单位，用尖括号包裹 | `<p>`、`<h1>` |
| **属性** | 标签的额外信息 | `lang="zh-CN"` |
| **开标签** | 标签的开始 | `<p>` |
| **闭标签** | 标签的结束 | `</p>` |
| **元素** | 完整的标签+内容 | `<p>这是文本</p>` |
| **DOCTYPE** | 文档类型声明 | `<!DOCTYPE html>` |

## 🤔 常见问题

**Q: 为什么需要闭标签？**
A: 闭标签告诉浏览器元素在哪里结束。例如 `<p>文本</p>` 中，闭标签 `</p>` 表示段落在这里结束。

**Q: 标签的顺序重要吗？**
A: 非常重要！标签必须按照嵌套的逻辑正确关闭。例如：
- ✅ 正确：`<html><body>内容</body></html>`
- ❌ 错误：`<html><body>内容</html></body>`

**Q: 是否所有 HTML 都需要 DOCTYPE？**
A: 是的。虽然有时浏览器可以在没有它的情况下工作，但 DOCTYPE 是必需的标准。
