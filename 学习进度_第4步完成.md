## 🎯 第 4 步的核心学习内容

### 链接 `<a>` 标签

| 类型 | href 值 | 示例 | 用途 |
|------|---------|------|------|
| 外部网站 | 完整 URL | `href="https://google.com"` | 链接到其他网站 |
| 同网站页面 | 相对路径 | `href="about.html"` | 网站内部导航 |
| 页面内跳转 | `#id` | `href="#section1"` | 跳转到页面的某个部分 |
| 邮件 | `mailto:` | `href="mailto:user@example.com"` | 发送邮件 |
| 电话 | `tel:` | `href="tel:+86138..."` | 拨打电话（手机） |

### 图片 `<img>` 标签

**必需属性：**
- `src` - 图片路径
- `alt` - 替代文本（非常重要）

**可选属性：**
- `width` - 宽度（像素）
- `height` - 高度（像素）
- `title` - 鼠标悬停提示

### 路径讲解

```
绝对路径：/images/photo.jpg （从网站根目录开始）
相对路径：images/photo.jpg  （从当前文件开始）
上级文件夹：../images/photo.jpg
```

## 🤔 常见问题解答

### Q: 为什么我的图片显示不出来？

**最可能的原因：**

1. **路径错误**
   ```html
   ❌ <img src="photo.jpg">  <!-- 但实际文件在 images 文件夹 -->
   ✅ <img src="images/photo.jpg">
   ```

2. **文件名错误（大小写）**
   ```html
   ❌ <img src="Photo.JPG">  <!-- 文件实际是 photo.jpg -->
   ✅ <img src="photo.jpg">
   ```

3. **网络图片 URL 错误**
   ```html
   ❌ <img src="https://example.com/photo.jpg">  <!-- 不能访问 -->
   ✅ <img src="https://valid-image-url.jpg">
   ```

4. **缺少 alt 属性（虽然图片会显示，但不规范）**
   ```html
   ❌ <img src="photo.jpg">
   ✅ <img src="photo.jpg" alt="照片描述">
   ```

### Q: 链接的下划线是怎么来的？

**浏览器的默认样式：**
- 未访问的链接：蓝色带下划线
- 已访问的链接：紫色带下划线

**这是正常的，以后用 CSS 可以修改链接的外观。**

### Q: 如何让链接在新标签打开？

```html
<a href="https://example.com" target="_blank">新标签打开</a>
```

### Q: `rel="nofollow"` 是做什么的？

```html
<a href="https://spam-site.com" rel="nofollow">不追踪的链接</a>
```

**作用：**
- 告诉搜索引擎不要跟踪这个链接
- 常用于：评论、广告、不信任的外部链接

## 💡 重要知识点总结

### 1. 链接的 3 个最重要的属性

```html
<!-- href - 链接的目标地址（必需） -->
<a href="...">

<!-- target - 打开方式 -->
<a href="..." target="_blank">

<!-- title - 鼠标悬停提示 -->
<a href="..." title="提示文本">
```

### 2. 图片的必需属性

```html
<!-- src - 图片路径 -->
<img src="...">

<!-- alt - 替代文本（非常重要，不能删除） -->
<img src="..." alt="图片描述">
```

### 3. 路径的黄金法则

```
相对路径 = 从当前文件的位置开始
绝对路径 = 从网站根目录开始

同一文件夹：photo.jpg
子文件夹：images/photo.jpg
上级文件夹：../photo.jpg
多个上级：../../photo.jpg
```
