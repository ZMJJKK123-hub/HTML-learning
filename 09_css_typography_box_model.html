✍️ CSS 第 9 步：排版与盒模型进阶

掌握文字排版细节（字体/行高/对齐/间距）与盒模型高级概念（margin 折叠、display 等）

1. 字体与字号（Font Family / Size）

字体族（font-family）

衬线字体（serif）：更正式，适合阅读型内容

无衬线字体（sans-serif）：现代、清爽，网页常用

等宽字体（monospace）：代码与对齐用

字号（px/em/rem/响应式）

0.875rem 小号文字

1rem 基准文字（随 html 调整）

1.25rem 稍大文字

clamp() 自适应标题

根据视口宽度平滑缩放的副标题

建议：用 rem 做排版基准，配合 clamp() 做自适应标题，统一视觉层级。
html { font-size: 16px } .title { font-size: clamp(1.25rem, 2.5vw + .5rem, 2rem) } .small { font-size: .875rem } /* 14px */

2. 行高、字重、字距与对齐

行高（line-height）

紧凑的行高（1.2），适合标题

宽松的行高（1.8），适合长段落阅读

字重 / 字距 / 对齐

常规权重（400）

半粗，增加字距（letter-spacing）

居中对齐（text-align: center）

两端对齐（justify）可以让段落左右对齐，但中英文混排要注意断行体验。

.lh-tight { line-height: 1.2 } .lh-relaxed { line-height: 1.8 } .weight-600 { font-weight: 600 } .spacing { letter-spacing: .04em } .align-justify { text-align: justify }

3. 文本修饰与换行控制

文本装饰

链接默认带下划线：示例链接

可通过 text-decoration / text-underline-offset 调整样式与位置。

换行与溢出

nowrap：这是一行很长很长很长很长很长的文字不会自动换行

break-word：averyveryverylongwordaveryveryverylongword 将在单词内部断开

.decoration a{ text-decoration: underline; text-underline-offset: 3px } .nowrap{ white-space: nowrap } .break-words{ overflow-wrap: break-word }

4. display：block / inline / inline-block

理解显示类型有助于正确布局与对齐：

block：独占一行，可设置宽高、内外边距
inline：不换行，宽高不可直接生效 inline-block：不换行，但可设置宽高
.block{ display:block } .inline{ display:inline } .inline-block{ display:inline-block }

5. margin 折叠（Margin Collapsing）

盒 A（上外边距 20px）
盒 B（上外边距 20px）

相邻块级元素之间的垂直外边距会发生折叠，取较大者，而非相加。

防止折叠的常见方式：在父容器开启 BFC（例如 `overflow: auto` 或 `display: flow-root`）。
.parent{ overflow:auto } /* 创建 BFC，阻止子元素与父元素之间的折叠 */ .flow-root{ display: flow-root } /* 现代语义化方案 */

6. 边框、圆角、阴影与轮廓

.box{ border:1px solid #e5e7eb; border-radius:8px; box-shadow:0 10px 25px rgba(0,0,0,.08) } .box:focus{ outline:3px solid rgba(37,99,235,.35) }

content（内容）

综合练习

  1. 为一段长文本设置合适的 `line-height` 与 `text-align: justify`，确保中英文混排观感舒适。
  2. 实现一个标题体系：`h1/h2/h3` 使用 `clamp()` 做响应式字号。
  3. 构建一个信息卡片：使用 `inline-block` 做标签行内布局，卡片 hover 时添加阴影与轻微提升。
  4. 创建一个父容器使其子元素不发生 margin 折叠（任选 BFC 技术）。
