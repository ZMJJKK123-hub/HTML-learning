变量与类型

let/const，模板字符串，解构。

const user = { name: 'Ada', level: 3 };
const scores = [92, 85, 77];
const { name } = user;
const [first] = scores;
const info = `${name} - first score ${first}`;

条件与循环

if/switch、for...of、数组迭代。

const passed = scores.filter(s => s >= 80);
const summary = passed.map((s, i) => `#${i + 1}: ${s}`).join(', ');

函数与作用域

函数声明/表达式/箭头，默认值，rest。

function greet(name = 'friend') {
  return `Hi, ${name}`;
}
const add = (a, b) => a + b;
const sumAll = (...nums) => nums.reduce((t, n) => t + n, 0);
