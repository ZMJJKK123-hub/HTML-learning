## 📋 文件详细说明

### 🎯 实践文件（按学习顺序）

**HTML 第 1-7 步**

| 文件名 | 用途 | 学习内容 |
|--------|------|---------|
| **01_hello_world.html** | 第一个网页 | DOCTYPE, html, head, body, meta |
| **02_page_structure.html** | 学习结构 | header, nav, main, section, aside, footer |
| **03_text_formatting.html** | 文本格式 | h1-h6, p, strong, em, code, 等所有文本标签 |
| **04_links_and_images.html** | 链接和图片 | a, img, 路径, 属性讲解 |
| **05_lists.html** | 列表演示 | ul/ol、嵌套列表、描述列表 |
| **06_tables.html** | 表格演示 | thead/tbody/tfoot、colgroup、合并单元格 |
| **07_forms.html** | 表单演示 | input/select/textarea、校验、可访问性 |
| **html_learn.html** | 完整参考 | 所有 HTML 标签的综合演示 |

**CSS 第 8-14 步**

| 文件名 | 用途 | 学习内容 |
|--------|------|---------|

| **13_css_variables_theming_cascade.html** | 变量与层叠 | tokens、CSS 变量、@layer、:where/:is、BEM |
