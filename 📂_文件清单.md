## 💡 文件使用技巧

### 1. 在 VS Code 中打开

```
1. 用 VS Code 打开 HTMLlearning 文件夹
2. 左侧会显示所有文件
3. 点击文件可快速打开
```

### 2. 在浏览器中查看

```
方法 1：双击 HTML 文件
方法 2：右键 → 用浏览器打开
方法 3：在 VS Code 中使用"Live Server"扩展
```

### 3. 用浏览器开发者工具学习

```
打开任何 HTML 文件 → 按 F12
   ↓
点击"检查元素"按钮
   ↓
选择网页上的任何元素
   ↓
看到对应的 HTML 代码
```

### 4. 快速搜索

```
在 VS Code 中：Ctrl+F 搜索
在 Markdown 文件中：Ctrl+F 搜索
在浏览器中：Ctrl+F 搜索
```
